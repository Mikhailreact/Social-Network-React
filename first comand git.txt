gitgit clone -  Клонировать Репозиторий (проект) к себе на компьютер!(change directory) cd - Жмем Tab открывается набор папок, выюираем свою и работаем в ней!  
git status - Статус текущего локального репозитория!
git add   -  Трекать фаил, тоeсть добавить фаил в локальный репозиторий! Личный твой, на твоем копьютере Git следит за твоим файлом!
git commit -am "added new file index.html" Зафексировали твой фаил
git commit -m "release v1" --- В кавычках пишем номер урока или релиз ну так чтобы понимать на какком месте
git push  -  Отпровляем изминения на сервер!
git pull  -  Вытащить изменения с сервера!
gitk --all& Открывает консоль где можно выбрать на каком этапе разработки тебе открыть проект или каккую версию открыть!

gitk --all&     (открыть графическую оболочку гита, чтобы позырить все коммиты)
ctrl + insert (копировать в буфер обмена) 
shift + insert (вставить из буфера обмена) 
git checkout commit-number (переключится на нужный коммит)
